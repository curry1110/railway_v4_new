# 軌道印記 v2.0 - 快速開始指南

## 🚀 5 分鐘快速部署

### 步驟 1：環境準備
```bash
# 確保已安裝 Node.js 和 MySQL
node --version  # 需要 v14+
mysql --version # 需要 v5.7+
```

### 步驟 2：安裝依賴
```bash
npm install
```

### 步驟 3：初始化資料庫
```bash
# 方式 A：使用 PowerShell（Windows）
Get-Content schema.sql | mysql -u root -p

# 方式 B：使用 CMD/Bash（Windows/Linux/Mac）
mysql -u root -p < schema.sql

# 輸入 MySQL root 密碼（預設：1234）
```

### 步驟 4：啟動伺服器
```bash
node server.js
# 輸出：Server running on http://localhost:3000
```

### 步驟 5：打開應用
在瀏覽器中訪問：`http://localhost:3000`

  - 重複進站：顯示「再次造訪 N 次」

- 站點資訊卡片顯示「造訪次數」
- 每次進站自動累加

## 🔧 常見問題

### Q1：無法連接資料庫？
**A**：檢查 `server.js` 第 15-23 行的資料庫設定：
```javascript
const pool = mysql.createPool({
  host: 'localhost',      // 改為你的主機
  user: 'root',           // 改為你的用戶名
  password: '1234',       // 改為你的密碼
  database: 'railway_imprint',
  // ...
});
```

### Q2：地圖不顯示？
**A**：
1. 確認瀏覽器支援 JavaScript
2. 檢查網路連接（需要下載 OpenStreetMap 地圖）
3. 打開瀏覽器開發者工具（F12）查看控制台錯誤

### Q3：模擬進站沒有反應？
**A**：
1. 確認已登入測試帳號 `testuser`
2. 確認伺服器正常運行
3. 檢查瀏覽器控制台是否有錯誤信息

### Q4：如何修改資料庫密碼？
**A**：
1. 修改 `server.js` 中的 `password` 欄位
2. 同時更新 MySQL 中的用戶密碼
3. 重啟伺服器
